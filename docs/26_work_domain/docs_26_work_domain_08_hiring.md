# Hiring Profile Specification

## Entity: HiringProfile (Snapshot)

Derived from Role version snapshot.

### Fields

-   role_version_reference
-   required_skills_snapshot (with minimum levels)
-   required_certifications_snapshot
-   optional_skills_snapshot
-   success_criteria_snapshot
-   kpi_expectations_snapshot
-   compensation_band_snapshot

## Purpose

-   Generates structured job advertisement
-   Drives interview scoring rubric
-   Preserves hiring baseline

## Rules

-   Snapshot at creation
-   Independent of future role changes
-   Used for job advertisement + interview scoring
